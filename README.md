## Install
1. Copy **synthwave.toml** to $HOME/.config/helix/themes/
2. Add `theme = "synthwave"` to $HOME/.config/helix/config.toml

[Optional]

Blue tab is the bufferline. To turn it on add to your config:
`editor.bufferline = "always"`
